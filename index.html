RethinkDB is an open-source, scalable database built for a world of realtime apps that push rather than pull.

      Works with modern stacks

      Get started by querying JSON documents with Python, Ruby, Node.js or dozens of other languages. Build a modern app using your favorite framework, paired with realtime technologies like Socket.io or SignalR.

      Robust architecture

      Immediate consistency, distributed joins and subqueries, aggregation and atomic updates.

      Fully open source

      RethinkDB is open source and built by a team around the world. Read the documentation, the getting started guide, etc.

      Scale your cluster with ease

      When you're ready to scale your app, shard and replicate in a few clicks using an intuitive web UI.

      If you need it, a simple API provides precise control over the cluster:

      r.table('games').configure(shards=5, replicas=3)

      Monitor your production cluster with live statistics and visibility into running jobs:

      r.db('system').table('jobs').changes()
