Build realtime apps

Event-driven architecture

Polling is obsolete. Get a firehose of changes from your database, or realtime incremental updates to query results.

  • Get realtime updates on all changes to your data.
  • Incremental query results and aggregations.
  • Filter and transform realtime streams of data.
Learn more →

Everything you need to build great apps

Store and query JSON documents with very little effort. Use a pleasant query language that supports queries like table joins and group by, and is easy to learn.

  • JSON data model and immediate consistency.
  • Distributed joins, subqueries, aggregation, atomic updates.
  • Secondary, compound, geospatial, and arbitrarily computed indexes.
  • Hadoop-style map/reduce.
Learn more →

Hassle-free operation at scale

Scale to multiple machines with very little effort. Use convenient API to shard and replicate in seconds, and have complete control of your cluster.

  • Simple cluster management, monitoring, and job control API.
  • Friendly web and command-line administration tools.
  • Sharding and replication to multiple nodes.
  • Queries are automatically parallelized and distributed.
